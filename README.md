# Project Asya

A private, elegant, and highly customized digital instrument designed to serve as a motivational dashboard. This repository is built as a "1:1 Pixel Perfect" visualization of academic brilliance, celebrating milestones, consistency, and growth seamlessly.

## 🌌 The Aesthetic: Midnight Luxe
- **Identity:** A private members' club meets a high-end watchmaker's atelier.
- **Palette:** Obsidian base, Champagne accents, and Ivory typography.
- **Micro-Interactions:** Magnetic buttons, staggered data reveals, and GSAP-powered ScrollTriggers for a fluid, weighted experience.

## ✨ Core Instrument Modules
1. **The Floating Island (Navbar):** A morphing, cohesive navigation pill adapting to the read-context.
2. **The Opening Shot:** An assertive hero entry with premium typography scaling.
3. **Interactive Functional Artifacts:**
   - *Diagnostic Shuffler:* Rotating insight cards.
   - *Telemetry Typewriter:* A live-feed typing analyzer.
   - *Cursor Protocol Scheduler:* An animated weekly grid micro-interaction.
4. **The Manifesto:** A dramatic juxtaposition of industry-standard vs elite focal points.
5. **Sticky Stacking Archive (Protocol):** 3 full-screen immersive cards layering dynamically on scroll.
6. **Hall of Fame & Weighted GPA Trajectory:** An integrated Recharts visualizer tracking exact, real-time historical cumulative performance. Only allows elite celebrations (4.5+ Average).

## 🛠 Technical Stack
- **React 19 (Vite):** Fast, modern DOM rendering.
- **Tailwind CSS v3:** Customized with Midnight Luxe tokens.
- **GSAP 3:** For ScrollTrigger pinning, entrance lifecycles, and micro-morphing.
- **Recharts & Lucide React:** For precise area graphing and iconography.

## 🚀 Local Development Setup

Clone the repository and spin up the development environment:

```bash
git clone https://github.com/quantumns/ProjectAsia.git
cd ProjectAsia
npm install
npm run dev
```

> **Note on Data Privacy:** The source code in this repository has been scrubbed of specific sensitive academic markers to maintain operational privacy while demonstrating the architectural capabilities of the codebase.
